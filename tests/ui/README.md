# UI tests (legacy location)

App-specific UI specs use **Option B (per-app dirs)**. They live under:

- **tests/crdc/** — CRDC Submission Portal (e.g. `crdc-homepage.spec.ts`)
- **tests/sts/** — STS (e.g. `sts-homepage.spec.ts`)

See `docs/plans/MULTI-PROJECT-EXTENSIBILITY-PLAN.md` for the layout convention.
